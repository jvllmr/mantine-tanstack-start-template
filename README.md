# mantine-tanstack-start-template

Minimal TanStack Start + Mantine + pnpm template

## Development

### Install dependencies

```shell
pnpm install --frozen-lock
```

### Start dev server

```shell
# Access via http://localhost:3000
pnpm dev
```

## Build for distribution

### Build

```shell
pnpm build
```

### Start built server

```shell
node .output/server/index.mjs
```
